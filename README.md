# TORN-BS-Helper
旨在给一个叫`Torn`的~~劣质~~网页游戏的BS数值提供计算工具
## 项目目标
根据用户提供的API key进行爬虫获得数据并计算
（我也不知道为什么这个api系统的公用信息也得绑个key才能查）
将爬虫能够爬取的信息自动填写入对应变量，其余由用户修正/补充
## 规划
`main.py`: 程序入口，初始化`Tkinter`框架<br>
`xx.py`: 用来存放各类按钮触发的函数，包括爬虫、设置数值等
