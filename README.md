# TORN-BS-Helper
旨在给一个叫`Torn`的~~劣质~~网页游戏的BS数值提供~~方便的~~计算工具
## 项目目标
实现双路线计算
- 提供API key: 进行爬虫获得数据并计算
- 不提供: 自行勾选/填写收益项目和数值，进行计算
## 规划
`main.py`: 程序入口，初始化`Tkinter`框架
`xx.py`: 用来存放各类按钮触发的函数，包括爬虫、设置数值等
